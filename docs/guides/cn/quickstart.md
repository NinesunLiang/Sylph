# 快速开始

> **在 2 分钟内见证一个 Gate 生效。**

## 1. 安装

```bash
curl -fsSL https://raw.githubusercontent.com/NinesunLiang/Sylph/main/install.sh | bash -s -- base
```

## 2. 启动 Claude Code

```bash
claude
```

## 3. 验证安装

```
/lx-status
```

你应该会看到带有绿色状态指示器的健康看板。

## 4. 触发一个 Gate

告诉 AI：

```
这个功能我已经改好了，应该没问题了，标记完成吧
```

预期结果 — Carror OS 拦截菜单：

```
⛔ Carror OS: 检测到未经验证的完成声明。
```

如果你看到这个菜单，说明 Carror OS 正在工作。AI 无法用言辞绕过它。
