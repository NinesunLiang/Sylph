# 面向新手

> **你不需要学习任何东西。Carror OS 静默运行。**

## 从这里开始

安装**基础版** — 这是所有初学者默认推荐的选择：

```bash
curl -fsSL https://raw.githubusercontent.com/NinesunLiang/Sylph/main/install.sh | bash -s -- base
```

## 你将得到什么

- **32 个物理 hook**，静默阻断危险 AI 行为
- **零学习成本**。继续像现在一样使用 Claude Code / OpenCode
- **自动质量门禁**。当你让 AI 提交时，Carror OS 在后台运行代码审查、安全扫描和风格检查
- **隐私保护**。AI 无法读取你的 `.env` 文件或向外部服务器发送凭据

## 为什么它对新手友好

基础版遵循一个原则：**先守护，后武装。**

- 它不会改变你的工作方式
- 它不需要阅读手册
- 它只在 AI 即将做出危险或不诚实的行为时介入
