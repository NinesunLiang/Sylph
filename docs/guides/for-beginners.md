# For Beginners

> **You do not need to learn anything. Carror OS works silently.**

## Start Here

Install the **Base Edition** -- it is the default recommendation for everyone starting out:

```bash
curl -fsSL https://raw.githubusercontent.com/NinesunLiang/Sylph/main/install.sh | bash -s -- base
```

## What You Get

- **32 physical hooks** that silently block dangerous AI behavior.
- **Zero new commands to learn.** Keep using Claude Code / OpenCode exactly as you do now.
- **Automatic quality gates.** When you ask the AI to commit, Carror OS runs code review, security scan, and style checks in the background.
- **Privacy protection.** The AI cannot read your `.env` files or send credentials to external servers.

## Why It Works for Beginners

Base Edition follows one principle: **guard first, arm later.**

- It does not change how you work.
- It does not require reading manuals.
- It only intervenes when the AI is about to do something dangerous or dishonest.
