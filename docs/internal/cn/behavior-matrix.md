
## 格式

| 场景 | 输入 | 期望输出 | 优先级 |
|------|------|---------|-------|
| 正常路径 | 合法输入 | 成功响应 | P0 |
| 边界值 | 临界输入 | 边界行为 | P1 |
| 错误路径 | 非法输入 | 错误响应 | P1 |
| 并发场景 | 并发请求 | 一致性 | P2 |

## 示例

| 场景 | 输入 | 期望 | 优先级 |
|------|------|------|-------|
| 正常登录 | 已注册手机+正确验证码 | 200 + token | P0 |
| 验证码过期 | 已注册手机+过期验证码 | 400 "code expired" | P1 |
| 手机号不存在 | 未注册手机 | 400 "user not found" | P1 |
| 频繁请求 | 1秒内>5次 | 429 "too many requests" | P2 |
