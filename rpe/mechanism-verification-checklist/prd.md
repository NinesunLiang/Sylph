# 机制验证清单

  ▎ 文档类型: RPE 验证规范
  ▎ 适用范围: CarrorOS 所有核心机制
  ▎ 验证原则: 文件存在 ≠ 机制存在 ≠ 机制有效

  ---
  📐 验证方法论

  验证层级金字塔

            实战效果验证
           ／           ＼
         运行测试验证
        ／              ＼
      逻辑自洽性验证
     ／                 ＼
   静态完整性检查

  四级验证标准

  ┌─────────┬───────────────┬──────────────────┬──────────┐
  │  级别   │   验证内容    │     通过标准     │ 证据类型 │
  ├─────────┼───────────────┼──────────────────┼──────────┤
  │ L1 静态 │ 文件/目录存在 │ 路径可访问       │ ls 输出  │
  ├─────────┼───────────────┼──────────────────┼──────────┤
  │ L2 逻辑 │ 文档定义完整  │ 有触发/流程/状态 │ 文档内容 │
  ├─────────┼───────────────┼──────────────────┼──────────┤
  │ L3 运行 │ 脚本可执行    │ 无错误退出       │ 执行日志 │
  ├─────────┼───────────────┼──────────────────┼──────────┤
  │ L4 实战 │ 真实场景工作  │ 产生预期结果     │ 工作产物 │
  └─────────┴───────────────┴──────────────────┴──────────┘

  状态标识

  - ⬜ 未验证 - 尚未开始
  - 🟨 部分验证 - 通过 L1-L2，未通过 L3-L4
  - 🟩 已验证 - 通过全部四级
  - 🟥 验证失败 - 存在缺陷

  引用完整性检查

  #!/bin/bash
  # 验证所有 @ 引用的文件存在

  echo "检查 AGENTS.md 引用完整性..."

  MISSING=0
  EMPTY=0
  VALID=0

  grep "^@" AGENTS.md | while read line; do
    file=$(echo "$line" | sed 's/@[[:space:]]*//')

    if [ ! -f "$file" ]; then
      echo "❌ 缺失: $file"
      ((MISSING++))
    elif [ ! -s "$file" ]; then
      echo "⚠️   空文件: $file"
      ((EMPTY++))
    else
      size=$(wc -c < "$file")
      echo "✅ $file ($size bytes)"
      ((VALID++))
    fi
  done

  echo ""
  echo "统计:"
  echo "  ✅ 有效: $VALID"
  echo "  ⚠️   空文件: $EMPTY"
  echo "  ❌ 缺失: $MISSING"
