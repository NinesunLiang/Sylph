## 用户纠正信号
用户说出以下任一内容 → 立即停止当前 Phase → 返回 Phase 1：
| 信号 | 含义|
|------|------|
|"这不是根源" / "That's not the root cause" | Why 分析偏离了方向|
|"别猜了" / "Stop guessing" | 你在用推测代替证据|
|"这个之前修过" / "This was fixed before" | 你遗漏了历史修复记录|
|"其他包也有这个问题" / "Other packages have this too" | 你的修复范围太窄 |

## 会话目标漂移检查
在每个 Phase 关卡处验证：- 会话目标锚点中的原始问题 = 当前调查目标- 若发生漂移 → 暂停 → 重新对齐 → 与用户确认 → 再继续- 绝不在漂移情况下继续，即使漂移方向"看起来更好"
