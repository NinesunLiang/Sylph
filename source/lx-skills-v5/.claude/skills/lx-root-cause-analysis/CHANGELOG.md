# root-cause-analysis 变更日志

## 适应性分析
本 skill 覆盖 Go 项目系统性 bug 的五层 Why 根因分析 + 免疫防护，涵盖：- 症状映射与时间线重建（Phase 1）- 断点隔离（Phase 2）- 五层 Why 证据链（Phase 3）- 根因消除与反模式拦截（Phase 4）- 免疫防护三重验证（Phase 5）
不适合拆为多个 skill 的原因：5 个 Phase 间有强数据依赖（故障链 → 断点 → Why 链 → 修复 → 免疫），拆分会导致跨 skill 传递大量上下文。

## v3.1 (当前)
