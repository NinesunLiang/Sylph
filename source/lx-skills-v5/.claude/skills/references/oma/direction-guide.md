# 方向指引

> 遵循 AGENTS.md §9 智力代理交互原则。每个 skill 交付后输出。
> 引用：`@references/oma/direction-guide.md`

## 模板

```
─── 方向指引 ───
📍 {当前阶段/位置}

{背景}

建议下一步:
  1. {命令} — 推荐 ✓ → {后果}
  2. {备选} → {后果}
  3. {备选} → {后果}
  4. 自定义操作 → 输入你想要的命令
  ─── 或直接输入你想要的命令 ───
```

## 原则

- 每个选项附带：命令 + 后果 + 推荐理由
- 末尾必有"自定义操作"兜底
- AI 推荐项标注 ✓
