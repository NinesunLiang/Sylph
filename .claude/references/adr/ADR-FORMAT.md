# ADR 格式模板

创建条件（三个全满足才写）：
1. **难撤销** — 后悔成本高
2. **非直觉** — 未来读者会问"为什么"
3. **真权衡** — 有明确替代方案并做了选择

## 文件命名

`NNNN-title-with-dashes.md`

N 从 0001 开始递增。标题用连字符小写。

## 模板

```markdown
# NNNN. 标题

**日期**: YYYY-MM-DD
**来源**: [会话/任务 ID]

## 上下文

为什么要做这个决策。

## 决策

我们选择了什么。

## 替代方案

考虑过的其他选项（最少 1 个）。

## 理由

为什么选这个而不是替代方案。

## 后果

这个决策带来的影响/约束。
```

> ADR 存在 `.claude/references/adr/`，通过 INDEX.md 自动索引加载。
