# danger-signals.md
# 异常信号检测清单（Pre-commit 阶段发现异常时加载）
# Referenced by: lx-pre-commit

## 编译异常
- [ ] 新引入的编译 warning（数量/类型）
- [ ] 增量编译 vs 全量编译结果不一致
- [ ] 跨平台编译差异

## 测试异常
- [ ] 新增测试 0 个（代码变更但无测试）
- [ ] 原有测试被修改而非新增
- [ ] 测试执行时间突变（>3x 原基准）
- [ ] Flaky test 模式（偶发通过/失败）

## 代码质量异常
- [ ] 单文件变更超过 200 行
- [ ] 新增代码行 > 删除代码行 5x
- [ ] 注释/日志/调试代码残留
- [ ] TODO/FIXME/HACK 标记新增
- [ ] 硬编码值（magic number/string）

## 依赖异常
- [ ] 新引入的外部依赖
- [ ] 依赖版本降级
- [ ] 间接依赖变更

## 范围异常
- [ ] 变更文件超出计划范围
- [ ] 修改了非目标模块的核心文件
- [ ] 新增了未计划的配置文件
