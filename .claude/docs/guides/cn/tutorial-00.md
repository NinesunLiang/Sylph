# T00 — 这是什么？

> 📍 教程链起点 | 下一篇：[T01 设计理由](tutorial-01.md) →

Carror OS 不是你装的一个插件。不是你写的一段 prompt。不是一个新的聊天工具。

**它是一个运行在 AI 工具之下的治理层。** 你继续用 Claude Code / OpenCode / Codex CLI 写代码。它在后台运行，在关键节点替你拦住那些最容易出问题的地方。

