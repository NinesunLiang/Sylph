## 🔥 核心价值：我们能为你解决什么？

### 💰 极致省钱：渐进式披露 (Token 瘦身)

我们彻底废除了将所有开发规范（代码风格、安全准则、Commit 格式）一股脑塞进 System Prompt 的落后做法。Carror OS 在底层构建了按需加载的引用库（每个 Skill 内的 `references/` 目录）：当你需要进行安全扫描时，它才加载安全规则；当你需要 `git commit` 时，它才加载团队提交规范。

**结果**：你的上下文永远保持极度干净，信噪比极高。不仅大幅削减了 API 账单，更让 AI 的"指令遵从率"产生了质的飞跃。

### 🛡️ 物理防毁坏：真实 Context 熔断 (OOM Hard-Gate)

大模型的注意力甜点区（Sweet-spot）通常在 40%-50%。超过 80% 就是灾难。Carror OS 的底层探针会实时测算你真实的 Token 消耗占比。

- **80% 物理锁死**：一旦真实 Token 超过 80% 危险线，底层探针会像拔掉电源一样直接抛出 `Exit 2`，**阻断一切写入和执行命令**！彻底杜绝末期幻觉毁坏代码库的惨剧。

### 🔒 数据绝对安全：企业级透明脱敏代理 (DLP Proxy)

**连大模型都看不到你的明文密码！**

`privacy-gate` 强行切断 AI 对 `.env`、`id_rsa`、`secret.yml` 等敏感文件的原生读取。Carror OS 提供了一个本地的脱敏金库（Vault）：AI 只能使用形如 `[MASKED_API_KEY]` 的占位符去修改配置或发起网络请求。底层的 Python 探针在命令落盘或发包前，会在本地静默替换为真实密钥；并在服务器返回结果时，进行二次混淆清洗。

**明文绝不出境，完美满足企业安全合规要求。**

### ⚔️ 打破"我写得全对"的幻觉：双终端交叉验证 (A→B→A Cross-Verify)

你不敢相信一个模型既写方案又做验收，对 AI 也是如此。当方案/报告完成后，系统会提示你另开一个新终端，**换一个不同模型**担任对抗性验收官，逐条审查断言的真实性。审查报告带回原终端进行交叉比对 — 在单机上实现了**跨模型攻防对抗 (Adversarial Cross-Verification)**。

对比一致才算验收通过，不一致则返回修订再循环。

## ⚡ 快速开始 (Quickstart)

Carror OS 支持**一键热插拔**到你的任意本地业务代码库中，用完即走，绝不污染全局环境变量。

我们深知新工具的认知负担有多重，因此提供了**渐进式交付 (Progressive Delivery)**。

### 第一步：打开你的终端

请先确保你的系统安装了 `curl` 或 `wget`：

```bash
# 进入你想要被 Carror OS 接管的业务项目根目录
cd @
```

### 第二步：选择你的守护级别

**🛡️ 选项 A：安装"基础版" (Base Edition)**

适合只想要 AI 变规矩、不犯错，不想学习任何新指令的你。安装后，45 个底层物理拦截器将在后台静默运行（防幻觉、防隐私泄露、80% Context OOM 熔断）。你只管正常让 AI 写代码，出错时它在底层默默扫除漏洞和幻觉。

```bash
curl -fsSL https://raw.githubusercontent.com/NinesunLiang/Sylph/main/install.sh | bash -s -- base
```

**⚔️ 选项 B：安装"增强版" (Enhanced Edition)**

适合接手复杂重构、大型特性开发、深水区 Debug 的资深工程师。解锁全部 24 款主动工作流流水线（RPE 工业级开发、双终端交叉验证对抗、DLP 脱敏代理等），让你"一人成军"。

```bash
curl -fsSL https://raw.githubusercontent.com/NinesunLiang/Sylph/main/install.sh | bash -s -- enhanced
```

> 注意：请将 URL 中的 `NinesunLiang/Sylph` 替换为你 Fork 后的真实 GitHub 仓库地址。

### 第三步：启动你的 AI IDE (Claude Code / OpenCode)

安装完成后，你的项目根目录下将生成 `AGENTS.md`（全平台主宪法）和 `.claude/`（内核探针与技能库）。现在，在对话框里唤醒你的 AI 助手：

```bash
# 如果你安装了增强版，你可以随时输入：
/lx-status
```

此时，终端会弹出一个炫酷的三屏健康看板，直观地向你展示**Token 节省账单**、**系统自愈率**和**任务执行画像**。
